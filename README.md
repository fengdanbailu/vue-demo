# vue-demo
目标：
```
1）从头开始开始构建一个vue项目，熟悉整体过程，过程主要参考https://github.com/PanJiaChen/vue-element-admin.git
2）封装基本组件，比如路由、导航栏、表格（支持前端分页，支持后端分页以及自定义一些样式）、图表、select（支持搜索）、tabpanel、form表单、弹窗（提示框，弹出新form表单）、等待图标、常量、util函数、ajax函数封装；
3）自定义组件；结合第三方兼容组件；
```

# 启动方式
```
yarn install
yarn dev
yarn build
```
